# 在AEM Forms中使用縮減器函式 — 圖表{#using-reducer-functions-in-aem-forms-charts}

Reducer函式用於彙總或分組顯示在圖表中的值。


例如，我們可以使用縮減器函式來顯示每個類別的交易數，或顯示以下範例資料中每個類別的最高費用

```json
[{
"category": "Groceries",
"amount": 10
},
{
"category": "Groceries",
"amount": 20
},
{
"category": "Travel",
"amount": 100
},
{
"category": "Gas",
"amount": 500
},
{
"category": "Gas",
"amount": 80
},
{
"category": "Groceries",
"amount": 30
},
{
"category": "Travel",
"amount": 200
}
]
```
