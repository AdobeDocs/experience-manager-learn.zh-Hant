# 在AEM Forms中使用減速器函式 — 圖表{#using-reducer-functions-in-aem-forms-charts}

減速器函式用於聚合或分組圖表中顯示的值。


例如，我們可以使用減速器函式來顯示每個類別的事務處理數，或從下面顯示的示例資料中顯示每個類別的最高費用

```json
[{
"category": "Groceries",
"amount": 10
},
{
"category": "Groceries",
"amount": 20
},
{
"category": "Travel",
"amount": 100
},
{
"category": "Gas",
"amount": 500
},
{
"category": "Gas",
"amount": 80
},
{
"category": "Groceries",
"amount": 30
},
{
"category": "Travel",
"amount": 200
}
]
```
