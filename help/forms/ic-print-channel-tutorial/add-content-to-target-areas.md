# 添加文本和影像內容以打印頻道文檔

在本部分，我們將將影像和文字內容新增至版面(xdp)中定義的目標區域。

現在，您已編寫了將保存收件人姓名和地址的第一個文檔片段，我們需要將文檔片段添加到打印渠道文檔中。

您可以將各種類型的內容（如文檔片段、影像、版面片段等）添加到打印渠道文檔。

影片會逐步帶您了解將內容新增至列印頻道檔案中適當目標區域的相關步驟。
