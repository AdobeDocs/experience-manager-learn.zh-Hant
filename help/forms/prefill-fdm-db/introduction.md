# 使用表單資料模型預先填寫表單

您將學習使用表單資料模型的request屬性來擷取資料以填入最適化表單。
在完成本課程後，您將學到以下內容。

* 建立支援RDBMS的表單資料模型
* 在2個實體之間建立關聯
* 在表格中填入表格資料模型 _的_ get服務傳回的資料
* 使用表單資料模型的請求屬性

## 必備條件

* AEM Forms的工作實例
* 熟悉MySQL資料庫和MySQL工作台
* 建立最適化表單的一些經驗
