# 使用表單資料模型預先填寫表單

您將學習使用表單資料模型的請求屬性來擷取資料以填入最適化表單。
在完成本課程後，您將學到以下內容。

* 建立支援RDBMS的表單資料模型
* 在2個實體之間建立關聯
* 在表格中填入&#x200B;_get_&#x200B;服務傳回的表格資料模型
* 使用表單資料模型的請求屬性
