# 新增文字和影像內容至網頁頻道檔案

在本部分，我們將新增影像和文字內容至標題面板。

既然您已編寫了第一個包含收件者名稱和地址的檔案片段，我們就必須將檔案片段新增至Web頻道檔案。

您可以將各種類型的內容（例如檔案片段、影像、版面片段等）新增至網頁頻道檔案。
