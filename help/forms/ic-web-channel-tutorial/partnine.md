# 將表添加到帳戶餘額面板{#adding-table-to-account-balance-panel}

在本部分，我們將向「帳戶餘額」面板添加一個表。

登入AEM Forms並導覽至「Adobe Experience Manager > Forms > Forms與檔案」。

開啟401KStatement資料夾。

在編輯模式下開啟401KStatement。

點選「帳戶餘額」目標區域，然後按一下「+」圖示以開啟「插入元件」對話方塊。

選擇表以插入表元件。

AEM Forms 6.4版已導入使用互動式通訊檔案中表格來顯示表格資料的功能。 以下影片會逐步說明在Web頻道檔案中設定表格的相關步驟。
