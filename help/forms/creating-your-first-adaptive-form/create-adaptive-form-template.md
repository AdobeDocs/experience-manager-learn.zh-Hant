# 建立最適化表單範本

每個最適化表單都以最適化表單範本為基礎。 範本定義結構、初始內容、主題等，最適化表單將繼承。 您可以建立新的最適化表單範本，或在建立最適化表單時使用現成可用的範本。
此處建立的範本將是日後最適化表單的基礎。
以下影片說明建立最適化表單範本的基本程式

>[!VIDEO](https://video.tv.adobe.com/v/37324/quality=9)
