# 簡介

在本教學課程中，您將瞭解如何實作使用提交的資料和附件預先填入&#x200B;**核心元件**&#x200B;型表單的簡單使用案例。

## 必要條件

* 使用具有核心元件的AEM Forms的體驗
* 在建置和部署AEM專案方面的經驗。
* 存取AEM Forms 6.5.19或更新版本。

若要在AEM Forms中實施上述使用案例，您需要下列專案

* AEM Forms 6.5.19執行個體或更高版本
* [Azure入口網站帳戶](https://portal.azure.com/)
