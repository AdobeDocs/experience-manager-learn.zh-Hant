# AEM工作流程中的規則編輯器

Or-Split元件用於在AEM Workflow中進行路由。 在AEM 6.5版之前，您必須編寫ECMA或自訂程式碼，才能啟用條件式路由。 在AEM 6.5中，我們現在可以使用規則編輯器來建立業務規則，以決定要執行的路由。
