# AEM Workflow中的規則編輯器

「Or分割」元件是用來在AEM Workflow中進行路由。 在AEM 6.5版本之前，您必須撰寫ECMA或自訂程式碼以啟用條件式路由。 在AEM 6.5中，現在可以使用規則編輯器建立商業規則，以決定要執行的路徑。
