# AEM工作流程中的檔案型別變數


檔案型別的變數可用來儲存記錄檔案和最適化表單附件。 這些變數隨後可用於傳送電子郵件元件，以透過電子郵件傳送附件。

本影片將反白顯示使用檔案型別變數來儲存產生的DoR，然後在AEM工作流程的傳送電子郵件步驟中使用檔案變數。
