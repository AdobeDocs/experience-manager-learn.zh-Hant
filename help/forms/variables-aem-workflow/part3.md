# 工作流中的文檔類型AEM變數


類型文檔的變數用於儲存記錄文檔、最適化表單附件。 然後，這些變數可用於「傳送電子郵件」元件，以透過電子郵件傳送附件。

此視訊會反白顯示使用檔案類型變數來儲存產生的DoR，然後在「工作流程」的傳送電子郵件步驟中使用檔案變AEM數。
