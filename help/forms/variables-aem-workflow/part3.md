# 工作流中的文檔類型AEM變數


類型文檔的變數用於儲存記錄文檔、自適應表單附件。 然後，這些變數可用於「發送電子郵件」元件中，以通過電子郵件發送附件。

此視頻將突出顯示使用文檔類型變數來儲存生成的DoR，然後在工作流的發送電子郵件步驟中使用文檔AEM變數。
