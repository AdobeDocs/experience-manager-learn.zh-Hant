# 具有JSON結構描述和資料的AEM Forms

多部分教學課程將逐步引導您完成使用JSON結構描述建立調適型表單和查詢已提交資料的相關步驟。

若要依照本教學課程進行，建議您安裝下列專案

* AEM 6.3或更新版本，搭配適當的Forms附加元件套件

* MySQL 8資料庫

* MySQL 8 Workbench

我們也建議您最好也對AEM和AEM Forms有一定的經驗。 您也應該熟悉安裝MySQL DB和MySQL Workbench
