# AEM FormsJSON架構和資料

多部分教程，引導您完成建立帶JSON架構的自適應表單和查詢提交資料所涉及的步驟。

要學習本教程，建議您安裝以下程式

* AEM 6.3或更高版本

* MySQL 8資料庫

* MySQL 8工作台

此外，還建議您在與AEM Forms方面有一定AEM的經驗。 您還應熟悉安裝MySQL資料庫和MySQL工作台
