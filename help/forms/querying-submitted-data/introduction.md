# AEM Forms與JSON結構描述與資料

多部分教學課程，可引導您逐步瞭解使用JSON結構描述建立最適化表單及查詢提交資料的相關步驟。

若要遵循本教學課程，建議您已安裝下列

* AEM 6.3或更新版本，以及適當的Forms addon套件

* MySQL 8資料庫

* MySQL 8工作台

我們也建議您對於和AEM Forms有一定的AEM經驗。 您也應該熟悉安裝MySQL DB和MySQL工作台
