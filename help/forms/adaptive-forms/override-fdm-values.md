# 覆寫表單資料模型屬性{#using-form-data-model-service-as-step-in-workflow}

在AEM Forms 6.5.13之前，您必須手動變更表單資料模型的雲端服務設定，以指向不同的端點，或建立全新的表單資料模型。

## 覆寫表單資料模型屬性

在AEM Forms 6.5.13之前，您必須手動變更表單資料模型的雲端服務設定，以指向不同的端點，或建立全新的表單資料模型。

AEM Forms 6.5.13引入覆寫表單資料模型屬性的功能，以便針對不同的服務端點輕鬆測試表單資料模型。
