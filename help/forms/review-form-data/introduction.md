# 提交前先檢閱表單資料

多部分教學課程，逐步引導您完成建立摘要元件的步驟，以在提交表單前檢閱和編輯表單資料。

![review-form-data](assets/review-form-data.png)

若要依照本教學課程操作，建議您安裝下列專案：

* AEM 6.5或更新版本，搭配適當的表單附加元件套件

我們也建議您最好在AEM和AEM Forms方面擁有一定程度的開發人員經驗。
