# 利用快速開發環境開發生命週期

瞭解使用RDE的開發/部署生命週期，以及它如何以速度提供價值。

以下視頻探討了基於RDE的從部署到開發的生命週期。 該視頻概述了用於有效使用RDE的關鍵子任務和開發流程。

>[!VIDEO](https://video.tv.adobe.com/v/3415492?quality=12&learn=on)


## RDE的優勢

RDE的主要好處是保持發展勢頭。

- 加快開發週期
- 減少停機時間
- 更好的測試
- 改進協作
- 提高生產效率
