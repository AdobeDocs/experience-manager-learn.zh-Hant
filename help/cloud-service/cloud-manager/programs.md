# 使用Adobe雲管理器 — 程式

雲管理器 **程式** 表示支援AEM邏輯業務計畫集的一組環境，通常對應於購買的服務級別協定(SLA)。 例如，一個方案可以代表支援AEM全球公共網站的資源，而另一個方案則代表一個內部中央資料庫。
