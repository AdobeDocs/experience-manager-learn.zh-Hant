# 使用Adobe Cloud Manager — 計畫

Cloud Manager **方案**&#x200B;代表支援邏輯業務方案集合的AEM環境集合，通常會對應到已購買的Service level agreement (SLA)。 例如，一個計畫可能代表AEM資源以支援全球公共網站，而另一個計畫代表內部中央DAM。
