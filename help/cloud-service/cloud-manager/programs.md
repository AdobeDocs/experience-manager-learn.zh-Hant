# 使用AdobeCloud Manager — 方案

Cloud Manager **Programs**&#x200B;表示支援業務計畫邏輯集的AEM環境集，通常對應於購買的服務級別協定(SLA)。 例如，一個方案可代表支援全域公用網站的AEM資源，而另一個方案則代表內部中央DAM。
