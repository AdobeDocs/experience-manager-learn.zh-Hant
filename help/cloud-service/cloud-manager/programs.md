# 使用Adobe雲管理器——程式

Cloud Manager **程式**&#x200B;表示支援邏輯業務計畫集的AEM環境集，通常對應於購買的服務級別協定(SLA)。 例如，一個方案可代表支AEM援全球公共網站的資源，而另一個方案則代表內部中央DAM。
