# 建立連線應用程式

連線應用程式是一種架構，可讓外部應用程式使用API和標準通訊協定（例如SAML、OAuth和OpenID Connect）與Salesforce整合。 連線的應用程式會使用這些通訊協定來驗證、授權並提供外部應用程式的單一登入(SSO)。
