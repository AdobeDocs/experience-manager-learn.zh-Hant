# 將AEM Forms與Adobe Analytics整合以報告表單資料欄位

瞭解如何使用AEM Forms標籤將Experience Platform as a Cloud Service與最適化表單上的Adobe Analytics整合。 此範例將引導您完成設定和實作步驟，以製作有關訪客如何與您的表單互動的深入報表。

## 先決條件

若要充分運用本教學課程，建議您符合下列必要條件：

* AEM Forms as a Cloud Service的部分使用體驗
* 存取Experience Platform標籤
* 存取Adobe Analytics

本教學課程使用AEM Forms內建的簡單調適型表單，並測量住處狀態值的表單提交專案，以及產生驗證錯誤的欄位。
