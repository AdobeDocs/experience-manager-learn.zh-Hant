# 建立Adobe SignAPI應用程式

您必須在Adobe Sign上建立API應用程式，才能與Adobe Sign平台整合。 API應用程式可代表用戶端應用程式，與Adobe Sign平台互動。 您可以在客戶或合作夥伴網域上建立API應用程式。
