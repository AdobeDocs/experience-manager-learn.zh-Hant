# 建立Adobe I/O專案

1. 建立整合的第一步是建立[Adobe ID](https://account.adobe.com/)
1. 前往[Adobe開發人員控制台](https://console.adobe.io/home)
1. 使用步驟1中建立的Adobe ID登入
1. 建立新專案
