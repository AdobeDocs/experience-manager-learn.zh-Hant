# 建立Adobe I/O項目

1. 建立整合的第一步是建立 [Adobe ID](https://account.adobe.com/)
1. 轉到 [Adobe Developer控制台](https://console.adobe.io/home)
1. 使用步驟1中建立的Adobe ID登錄
1. 建立新項目

請按照視頻建立Adobe I/O項目。
