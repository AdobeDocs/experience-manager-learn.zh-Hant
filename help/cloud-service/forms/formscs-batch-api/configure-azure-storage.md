# 簡介

Azure儲存帳戶包含所有Azure儲存資料對象，包括Blob、檔案共用、隊列、表和磁碟。 儲存帳戶為您的Azure儲存資料提供唯一的命名空間，可透過HTTP或HTTPS從世界上的任何地方存取。 您的儲存帳戶中的資料經久耐用，可高度使用，安全且可擴展。
