# 簡介

Azure儲存帳戶包含所有Azure儲存資料對象，包括blob、檔案共用、隊列、表和磁碟。 儲存帳戶為您的Azure儲存資料提供了唯一的命名空間，該命名空間可通過HTTP或HTTPS從世界上的任何位置訪問。 您的儲存帳戶中的資料具有持久性、高可用性、安全性和大規模擴展性。
