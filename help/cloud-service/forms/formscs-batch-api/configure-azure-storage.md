# 簡介

Azure儲存體帳戶包含您的所有Azure儲存體資料物件，包括blob、檔案共用、佇列、表格和磁碟。 儲存體帳戶為您的Azure儲存體資料提供獨特的名稱空間，您可在全球任何地方透過HTTP或HTTPS存取。 儲存帳戶中的資料經久耐用、高度可用、安全，而且可大幅擴充。
