# 簡介

Azure儲存體帳戶包含您的所有Azure儲存體資料物件，包括blob、檔案共用、佇列、表格和磁碟。 儲存體帳戶為您的Azure儲存體資料提供獨特的名稱空間，可透過HTTP或HTTPS從全球任何地方存取。 儲存帳戶中的資料經久耐用、高可用性、安全，並可大規模擴充。
