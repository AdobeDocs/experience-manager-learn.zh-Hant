# 垂直索引標籤簡介

垂直標籤可用來以精簡、易於導覽的格式整理和顯示內容。 垂直提供下列優點
1. 空間效率 — 垂直索引標籤利用垂直空間，當要擷取大量資訊時，這樣會更有效率。
1. 改良的導覽功能 — 可讓使用者輕鬆在表單的不同區段之間快速切換。
1. Visual Clarity — 將導覽選項集中在一處，有助於維持簡潔的版面。
