# 使用批量導入

AEMas a Cloud Service中的「大量匯入」工具可讓管理員以安全且有效的方式，從雲端儲存體(Azure Blob儲存體或Amazon S3)大量匯入資產。

>[!VIDEO](https://video.tv.adobe.com/v/329680/?quality=12&learn=on)

## 排程大量匯入

「大量匯入」支援排程執行設定，包括：

+ 在定義的日期和時間執行一次
+ 每小時、每天或每週執行一次
