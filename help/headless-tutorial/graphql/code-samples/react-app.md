# React應用程式

使用簡單的React應用程式探索AEM無周邊GraphQL API。 此React應用程式會連線顯示WKND歷險記清單，並提供每個歷險的詳細檢視。

此程式碼：

+ 連接到 [wknd.site](https://wknd.site)的AEM Publish服務，且不需要驗證
+ 使用持續的查詢： `wknd-shared/adventures-all` 和 `wknd-shared/adventures-by-slug`
