# Campaign與Target整合

{{campaign-description}}

{{target-description}}

使用Adobe Campaign搭配Adobe Target來個人化和最佳化您的電子郵件內容。 主要優點包括：

+ **Adobe Campaign資料**：使用Adobe Campaign Data Mart增強Adobe Target中的鎖定目標。
+ **對象細分**：將區段連結至訪客ID，以最佳化鎖定目標和個人化體驗。
+ **目標定位增強功能**：存取寶貴資料（使用者代理、IP位址、地理位置），以改進目標定位和促銷活動。
