# 在AEM中使用收件匣

AEM的「收件匣」可協助您進一步協作，並洞悉工作和工作流程。 「收件箱」的「日曆視圖」在基於時間的視圖中跟蹤項目和任務，以便更輕鬆地進行項目管理和計畫。 使用「日曆」檢視，使用者可以整體檢視目前進行中、已規劃、即將到期和過期的專案和工作。
