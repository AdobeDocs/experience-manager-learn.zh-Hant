# 在AEM中使用收件匣

AEM收件匣可促進更多共同作業，並可洞察工作和工作流程。 收件箱的日曆視圖以基於時間的視圖跟蹤項目和任務，以便更輕鬆地進行項目管理和計畫。 透過「日曆」檢視，使用者可全面掌握目前進行中、已規劃、即將到期及過期的專案和工作。
