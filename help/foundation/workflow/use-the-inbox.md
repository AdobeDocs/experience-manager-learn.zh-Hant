# 使用收件箱AEM

收件AEM箱有助於更好地協作和查看任務和工作流。 「收件箱」的「日曆視圖」在基於時間的視圖中跟蹤項目和任務，以便更輕鬆地進行項目管理和計畫。 使用「日曆」視圖，用戶可以全面查看當前正在進行、計畫、即將到期和過期的項目和任務。
