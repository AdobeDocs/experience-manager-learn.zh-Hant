# 使用工作流程編輯器

工作流程會在Experience Manager中啟用業務流程管理，並用於自動處理內容，以及促進需要人為決策的治理和流程。 工作流程是透過由一系列步驟組成的工作流程模型來定義，並在[!UICONTROL 工作流程模型編輯器]中建立和管理。
