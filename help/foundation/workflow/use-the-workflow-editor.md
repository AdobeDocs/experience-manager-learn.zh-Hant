# 使用工作流程編輯器

工作流程可啟用Experience Manager的業務程式管理，並用於自動處理內容，以及促進需要人為決策的治理和程式。 工作流程是透過「工作流程模型」來定義，這些模型由一系列步驟組成，並在中建立和管理 [!UICONTROL 工作流程模型編輯器].
