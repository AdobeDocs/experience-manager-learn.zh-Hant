# 使用工作流編輯器

工作流支援Experience Manager中的業務流程管理，並用於內容的自動處理以及方便需要人為決策的治理和流程。 工作流通過由一系列步驟組成的工作流模型定義，並在 [!UICONTROL 工作流模型編輯器]。
