# 使用工作流程編輯器

工作流程可讓Experience Manager中的業務流程管理，並用於自動處理內容，以及促進需要人為決策的治理和流程。 工作流通過工作流模型定義，工作流模型由一系列步驟組成，並在[!UICONTROL 工作流模型編輯器]中建立和管理。
