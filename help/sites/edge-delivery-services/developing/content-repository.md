# 內容存放庫

若要將Adobe Experience Manager (AEM)與Edge Delivery Services整合，請先將[SharePoint](#sharepoint)或[Google Drive](#google-drive)設定為您的內容存放庫。 兩個平台都可與AEM搭配使用，提供順暢的內容管理和傳送。 選擇符合您偏好設定的專案，並依照步驟操作，使其啟動並與AEM和Edge Delivery Services搭配執行。

## 檔案製作選項

[檔案製作](../../document-authoring/set-up.md)、[SharePoint](#sharepoint)和[Google磁碟機](#google-drive)有兩個支援的內容存放庫。
