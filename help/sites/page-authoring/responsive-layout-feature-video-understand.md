# 瞭解回應式佈局 {#understanding-responsive-layout-with-aem-sites}

Adobe Experience Manager Sites可讓作者根據裝置寬度管理元件的寬度，以促進回應式網站設計。 使用佈局模式和佈局容器，使用者可以為不同裝置建立回應式內容並預覽一般使用者體驗，而無需離開AEM UI。
