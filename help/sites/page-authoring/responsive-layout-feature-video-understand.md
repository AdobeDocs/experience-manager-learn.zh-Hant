# 瞭解響應佈局 {#understanding-responsive-layout-with-aem-sites}

Adobe Experience Manager Sites允許作者基於設備寬度管理元件的寬度以便於響應性的站點設計。 使用佈局模式和佈局容器，用戶可以為不同設備建立響應內容並預覽最終用戶體驗，而無需離開AEMUI。
