# 了解回應式版面{#understanding-responsive-layout-with-aem-sites}

Adobe Experience Manager Sites可讓作者根據裝置寬度管理元件的寬度，以便進行回應式網站設計。 透過「版面模式」和「版面容器」，使用者可針對不同裝置建立回應式內容，並預覽一般使用者體驗，而不需離開AEM UI。
