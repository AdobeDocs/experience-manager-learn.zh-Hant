# 創作頁面 {#authoring-page-aem-sites}

瞭解如何使用網站編輯器的UI在Adobe Experience Manager Sites建立頁面。

>[!VIDEO](https://video.tv.adobe.com/v/31828?quality=12&learn=on)

在此視頻中，包括以下功能：

* 基於模板建立頁面
* 介紹頁面的不同創作模式
   * 編輯
   * 配置
   * 開發人員
   * Timewarp
* 頁面編輯器功能
   * 元件查找器
   * 資產尋找器
* 將內容添加到現有元件
* 將新元件添加到頁面
* 頁面屬性
   * 發佈頁面
   * 取消發佈頁面
   * 編輯範本
   * 以已發佈狀態檢視
