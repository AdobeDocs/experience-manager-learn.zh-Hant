# 編寫頁面 {#authoring-page-aem-sites}

{{edge-delivery-services-and-page-editor}}

瞭解如何使用網站編輯器的UI在Adobe Experience Manager Sites中編寫頁面。

>[!VIDEO](https://video.tv.adobe.com/v/31828?quality=12&learn=on)

在本影片中，將介紹下列功能或特性：

* 根據範本建立頁面
* 為頁面匯入不同的撰寫模式
   * 編輯
   * 配置
   * 開發人員
   * Timewarp
* 頁面編輯器功能
   * 元件尋找器
   * 資產尋找器
* 將內容新增至現有元件
* 將新元件新增至頁面
* 頁面屬性
   * 發佈頁面
   * 取消發佈頁面
   * 編輯範本
   * 以已發佈狀態檢視
