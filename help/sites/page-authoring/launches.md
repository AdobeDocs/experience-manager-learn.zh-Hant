# 啟動

{{edge-delivery-services-and-page-editor}}

AEM Sites中的啟動提供一種建立、編寫和檢閱網站內容以供未來發行的方式。 在啟動期間，生產網站可以持續進化並逐日改變，就像平常一樣。
