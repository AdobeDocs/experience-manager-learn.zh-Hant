# 將Dynamic Media與AEM Sites核心元件一起使用{#dynamic-media-sites-core-components}

映像元件是Experience Manager核心元件的一部分，它內置了對Dynamic Media的支援。 影像元件可讓內容作者使用Dynamic Media的功能，例如AEM Sites頁面上的影像預設集、智慧型裁切和影像修飾元。
