# 將Dynamic Media與AEM Sites核心元件配合使用 {#dynamic-media-sites-core-components}

映像元件是Experience Manager核心元件的一部分，它內置了對Dynamic Media的支援。 「影像」元件允許內容作者在AEM Sites頁面上使用Dynamic Media的功能，如「影像預設」、「智慧裁剪」和「影像修飾符」。
