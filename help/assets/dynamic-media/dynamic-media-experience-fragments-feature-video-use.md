# 搭配AEM Assets Dynamic Media使用體驗片段{#using-experience-fragments-with-aem-assets-dynamic-media}

AEM Assets Dynamic Media透過互動式媒體檢視器整合資產熱點與體驗片段。 行銷人員可以使用文字、影像和連結來製作體驗片段，以填入自訂快顯視窗，並將該快顯視窗與橫幅、輪播或影片上的熱點建立關聯。
