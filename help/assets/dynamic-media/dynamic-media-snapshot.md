# Dynamic Media快照 {#snapshot}

Dynamic Media快照是一個直觀的演示工具，旨在展示Dynamic Media在優化和動態影像傳輸方面的威力。 對test影像或Dynamic MediaURL進行實驗，以直觀地觀察各種Dynamic Media影像修飾符的輸出，並對檔案大小（使用WebP和AVIF傳遞）、網路和設備像素比(DPR)進行智慧成像優化。
