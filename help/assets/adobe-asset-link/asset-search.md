# 資產搜尋{#asset-search}

使用「Adobe資產連結」面板，創意使用者可以使用關鍵字搜尋儲存在AEM Assets的資產，或在特定位置下執行搜尋。 除了基本搜尋選項外，「Adobe資產連結」還可讓您使用資產在AEM Assets內搜尋視覺上類似的結果。 視覺搜尋功能由Adobe Sensei的機器學習提供支援，可顯示與目標影像相關的美學特徵結果。
