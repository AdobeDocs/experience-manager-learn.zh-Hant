# 資產搜尋 {#asset-search}

透過Adobe資產連結面板，創意使用者可以使用關鍵字搜尋AEM Assets中儲存的資產，或在特定位置下執行搜尋。 除了基本搜尋選項外，「Adobe資產連結」也可讓您使用資產，在AEM Assets中以視覺化方式搜尋類似結果。 視覺搜尋功能採用Adobe Sensei的機器學習技術，可顯示具有與目標影像相關之審美特徵的結果。
