# 登入登出{#check-in-check-out}

Adobe資產可讓您直接從您正在使用的創意應用程式中查看AEM Assets，並立即開始編輯。 在您的Creative Cloud應用程式中編輯完資產後，您可以使用適當的版本修訂和注釋，輕鬆將資產存回AEM Assets。
