# AEM Assets 影片和教學課程 {#overview}

{{edge-delivery-services}}

Adobe Experience Manager (AEM) Assets 是 AEM 平台上的數位資產管理工具，讓使用者可以在網頁型存放庫中建立、管理及共用其數位資產 (影像、影片、文件和音訊片段)。本使用手冊包含 AEM Assets 許多特性與功能的相關影片和教學課程。
