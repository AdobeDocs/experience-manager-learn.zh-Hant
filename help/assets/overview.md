# AEM Assets 影片和教學課程 {#overview}

Adobe Experience Manager(AEM)Assets是AEM Platform上的數位資產管理工具，可讓使用者在網頁型存放庫中建立、管理和共用其數位資產（影像、影片、檔案和音訊剪輯）。 本使用手冊包含有關AEM Assets許多功能的影片和教學課程。
